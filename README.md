# Knowledge Dash

Standalone static version of Knowledge Dash.

Open `index.html` in a browser or publish the folder to GitHub Pages.

Sections:
- `index.html` - main screen
- `game.html` - playable rhythm platformer
- `levels.html` - level overview
- `collection.html` - localStorage collection
- `controls.html` - controls and rules

No Lovable runtime is required.
